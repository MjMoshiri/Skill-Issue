# MLGA — Make Laptop Great Again

My personal collection of Claude Code skills, agents, and prompts. The setup that makes my laptop actually useful.

Public so I can share configs without rebuilding context every time someone asks "what's that prompt you use?"

## What's in here

- `agents/` — Claude Code subagents I use day to day

## Use it

Drop the files into `~/.claude/agents/` (or your project's `.claude/agents/`). They show up as agent types you can spawn.

No license, no warranty. Take what works, ignore what doesn't.
