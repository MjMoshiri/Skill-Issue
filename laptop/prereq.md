# Laptop prerequisites

Just what I want. Install however; no guide here.

## Must have

- Homebrew (macOS)
- Ghostty
  - font: MesloLGS NF (or other Nerd Font)
  - shell: zsh
- Powerlevel10k (Rainbow-style is fine)
- Raycast
- Rectangle
- Grok CLI
- Claude CLI
  - RTK PreToolUse hook on Bash (`rtk hook claude`)
  - no RTK skill / no `RTK.md` / no `@RTK.md`
- ctx7 (Context7 CLI)
- RTK
  - proxy by default (agent hook + interactive shell rewrite)
  - **do not** install the RTK skill
